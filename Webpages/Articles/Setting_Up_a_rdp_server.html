Setup

First, make sure to have a bootable drive (in my case debian)

Continue doing a standard setup, make sure ssh is enabled

Making it a Static IP

Do "sudo nano /etc/networks/interfaces" and add the following:

{wifi interface} auto
iface {wifi interface} inet static
    address {desired ip address}
    netmask {desired ip netmask}
    gateway {desired ip gateway}

Replace {wifi interface} with the wifi interface you get by doing "ip addr"

Adding rdp

This RDP is ONLY for in your local network and is unencrypted (since i am too lazy to do anything with TSL)

Install the following packages

sudo apt-get install -y Xorg
